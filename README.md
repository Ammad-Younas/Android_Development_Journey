# Android & Kotlin Development Journey

A collection of projects built while learning **Android development** and **Kotlin** — ranging from console-based programs to full Android applications.  
More projects will be added as the journey continues.

---

## Projects

| # | Project | Type | Description |

| 2 | [CoffeeMachine](./CoffeeMachine) | Kotlin Console | Models an advanced coffee machine with drink selection and ingredient management. |

## Tech Stack

- **Language:** Kotlin
- **Android SDK** (for mobile projects)
- **Jetpack Compose** (where applicable)
- **IntelliJ IDEA / Android Studio**

---

## Getting Started

### Kotlin Console Projects
Open the project folder in **IntelliJ IDEA** and run `Main.kt`.

### Android Projects
1. Open the project folder in **Android Studio**.
2. Let Gradle sync finish.
3. Run on an emulator or a physical device.
